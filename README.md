# 💰 Expense Tracker Web App

A simple and efficient Expense Tracker web application built using Python (Flask), HTML, and CSS.
This application allows users to manage their daily expenses with real-time updates and persistent data storage.

## 🚀 Features

- ➕ Add new expenses
- 📋 View all expenses
- ❌ Delete individual expenses
- 🧹 Clear all expenses
- 💰 Automatic total calculation
- 💾 Data stored using JSON file (persistent storage)

## 🛠️ Technologies Used

- Python (Flask)
- HTML
- CSS
- JSON (for data storage)

## ▶️ How to Run the Project

1. Clone the repository:
2. git clone https://github.com/rajinipanasa/expenses-tracker.git

2. Navigate to the project folder:

cd expenses-tracker

3. Install Flask:

pip install flask


4. Run the application:

python main.py


5. Open your browser and go to:

http://127.0.0.1:5000/

## 📌 Purpose

This project demonstrates basic full-stack development using Flask. 
It includes backend logic, frontend design, and file-based data storage using JSON.
